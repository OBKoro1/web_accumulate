## 前端算法

可以跟着本文档的栗子来慢慢学习一下算法，对于提高自身的编程和逻辑能力也是相当有效的一个途径。

文档会将算法按照**难度分级**，代码中都有**详细注释**，且会提供**多种解法**。

::: tip 
重要的是解题的思路，扩展我们的脑洞，不要拘泥于应试！
:::
