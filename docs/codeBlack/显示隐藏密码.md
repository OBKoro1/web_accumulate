### 代码：

**超简单，就是切换input框的type类型:**

```js
function isShow() {
    let input = document.getElementById("inputId"); // input的dom
    if (input.type == "password") {
        input.type = 'text'; // 显示
    } else {
        input.type = 'password'; // 隐藏
    }
}
```
